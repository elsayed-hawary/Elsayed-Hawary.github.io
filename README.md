# Моя Семья - Семейный менеджер

Приложение для управления семейными делами: задачи, покупки, расходы, расписание и события.

## 📱 Возможности

- **Управление семьей** - создание семьи и приглашение участников
- **Задачи** - создание и назначение задач членам семьи
- **Покупки** - список покупок с отслеживанием статуса
- **Расходы** - учет семейных расходов с категориями
- **Расписание** - личное расписание для каждого члена семьи
- **События** - важные даты и события семьи
- **Статистика** - визуальное отображение активности
- **Отчеты** - ежемесячные отчеты о расходах
- **Уведомления** - система оповещений

## 🚀 Запуск

1. Откройте ссылку GitHub Pages
2. Зарегистрируйтесь или войдите с помощью Apple/Google
3. Создайте семью и пригласите участников
4. Начинайте управлять семейными делами

## 💾 Хранение данных

Все данные сохраняются в localStorage вашего браузера. Данные не передаются на внешние серверы.
